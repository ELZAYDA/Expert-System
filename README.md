# 🫀 Expert System for Cardiovascular Disease Risk Assessment

## نظام خبير لتقييم مخاطر أمراض القلب والأوعية الدموية

---

## 📋 نظرة عامة عن المشروع

هذا المشروع هو **نظام خبير (Expert System)** متكامل يقوم بتقييم مخاطر أمراض القلب لدى المرضى باستخدام:

1. **قواعد معرفية (Knowledge Base)** مستمدة من الإرشادات الطبية العالمية (AHA/ACC)
2. **شبكة عصبية اصطناعية (Neural Network)** تتعلم الأنماط من البيانات
3. **محرك استدلال هجين (Hybrid Inference Engine)** يجمع بين النظامين
